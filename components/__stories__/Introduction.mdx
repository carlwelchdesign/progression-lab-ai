## What is ProgressionLab.AI?

ProgressionLab.AI is an intelligent chord progression generator designed for musicians and producers. It helps you discover unique and inspiring chord progressions using AI-powered suggestions, visual feedback through guitar and piano diagrams, and real-time audio playback.

### Key Features

- **AI-Powered Chord Generation**: Get intelligent chord progression suggestions based on genre, mood, and harmonic structure
- **Multi-Instrument Support**: Guitar and piano diagrams for immediate visual reference
- **Real-Time Audio Playback**: Hear your progressions instantly with MIDI playback and effects
- **Progression Sharing**: Save and share your favorite progressions with others
- **Dark Theme UI**: Beautiful, professional interface optimized for music production workflows

## Tech Stack

ProgressionLab.AI is built with modern web technologies:

- **Next.js 15** — React framework with server and client components
- **React 19** — Modern UI library with latest features
- **TypeScript** — Type-safe JavaScript for robust code
- **Material-UI (MUI) v7** — Comprehensive component library
- **Storybook 10** — Component documentation and testing
- **Vitest** — Unit testing framework
- **Prisma** — Database ORM
- **MSW** — API mocking for development and testing
- **MIDI.js** — Audio synthesis and playback
